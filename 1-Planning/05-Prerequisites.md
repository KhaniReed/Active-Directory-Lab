# Prerequisites

- Minimum 16 GB RAM (recommended for smooth performance)
- At least 50 GB of free disk space
- Multi-core CPU (Intel i5 or equivalent)

---

## 🖥️ Software Requirements

### Virtualization Software
- Oracle VM VirtualBox or VMware Workstation (I am utilizing VMware for my lab) 

---

### Operating System ISOs
- Windows Server 2022 ISO
- Windows 11 ISO

---

## 🌐 Networking Requirements
- Ability to configure internal or host-only networks
- Basic understanding of IP addressing and DNS

---

## 🧠 Knowledge Prerequisites
- Basic understanding of:
  - Networking (TCP/IP, DNS)
  - Windows operating systems
  - Command line (PowerShell / CMD)
  - Active Directory and GPO concepts
