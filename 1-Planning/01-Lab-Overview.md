# Lab Overview

## 📌 Introduction
This project is a hands-on Active Directory home lab designed to simulate a real-world enterprise environment using Windows Server 2022. The lab focuses on building, configuring, and securing an Active Directory Domain Services (AD DS) infrastructure while documenting each step in a structured and reproducible format.

This lab is intended to strengthen practical skills in system administration, cybersecurity, and enterprise network management.

---

## 🧠 Purpose
The purpose of this lab is to:
- Gain hands-on experience with Active Directory
- Understand domain-based network architecture
- Practice implementing enterprise security controls

---

## 🏗️ What This Lab Includes
- Deployment of a Windows Server 2022 Domain Controller
- Domain creation and configuration
- Client machine integration into the domain
- User, group, and Organizational Unit (OU) management
- Group Policy Object (GPO) implementation
- Security hardening (firewall, network, and services)
- PowerShell automation scripts
- Validation and troubleshooting documentation
